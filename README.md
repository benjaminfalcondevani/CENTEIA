# CENTEIA Consulting — I CAN

Página estática que unifica el mensaje visual del marco **I CAN** para explicar cómo avanzar con Inteligencia Artificial: identificar oportunidades, comprender el contexto, actuar con soluciones responsables y nutrir la mejora continua.

## Archivos

- `index.html`: estructura semántica del póster/landing en español.
- `styles.css`: sistema visual responsive con gradientes, tarjetas de pilares y cierre de mensaje.

## Vista local

Abre `index.html` en un navegador o sirve el directorio con:

```bash
python3 -m http.server 8000
```
